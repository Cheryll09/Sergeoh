How It Works
Trailing Stop Formula
Stop = ATH × (1 − (ATR₄₂ / Close) × N)
where N = ATR multiplier (default 10)
Stop only moves UP (ratchet mechanism)
SELL if Close < Trailing Stop
Position Sizing Formula
w = (VolTarget% / σ₄₂ₐₙₙ) × (1 / max(MinDiv, N))
Shares = (w × AUM) / Price
VolTarget = 30% (default)
σ = 42-day annualised volatility
